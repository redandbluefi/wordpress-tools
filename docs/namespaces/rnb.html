get¶

Prints a template. Templates can be just about anything, but they must have a function with the name you use to print the template. Sample template: ``` <?php function sample_template($title = 'Hello', $content = 'you') { ?> <div> <h1><?=$title?></h1> <p><?=$content?></p> </div> <?php } ```

package

Default

string

Relative path from theme root, usually templates/file.php
